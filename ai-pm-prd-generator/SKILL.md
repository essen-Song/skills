# AI产品经理PRD生成器

## 技能概述

这个Skill专门为AI产品经理设计，帮助您快速将模糊的产品想法转化为可以直接喂给AI开发工具（Claude Code、Cursor等）执行的一页纸PRD。

## 使用场景

- 当您有一个产品想法但不知道如何开始写PRD时
- 当您需要快速为vibe coding项目创建执行文档时
- 当您想要系统化产品思考流程时

## PRD生成流程

### 第一步：产品定义
引导用户明确：
- **产品名称**：简洁有辨识度的名称
- **一句话描述**：用一句话说清楚产品是什么
- **目标用户**：具体的使用人群
- **核心价值**：解决什么痛点

### 第二步：功能规划
帮助用户梳理：
- **核心功能**：MVP必须包含的功能
- **用户流程**：用户如何使用产品
- **技术选型**：建议的技术栈
- **界面要求**：基本的UI/UX要求

### 第三步：约束条件
明确边界：
- **不做功能**：明确排除的范围
- **技术限制**：技术上的约束条件
- **时间要求**：开发时间预期

### 第四步：输出格式
生成标准PRD模板：
- 一页纸格式，适合AI阅读
- 结构化内容，便于AI理解
- 具体可执行的开发指令

## 输出示例

```markdown
# [产品名称] PRD

## 产品概述
- **一句话描述**: [产品描述]
- **目标用户**: [用户群体]
- **核心价值**: [解决什么问题]

## 核心功能
1. [功能1描述]
2. [功能2描述]
3. [功能3描述]

## 技术选型
- 前端: [技术栈]
- 后端: [技术栈]
- 数据库: [技术栈]

## 用户流程
1. 用户进入 → [步骤1]
2. 用户操作 → [步骤2]
3. 用户完成 → [步骤3]

## 约束条件
- **不做**: [明确排除的功能]
- **技术限制**: [技术约束]
- **时间**: [开发周期]

## AI开发指令
[具体的代码开发指令]
```

## 使用方式

当您想要创建一个新的vibe coding项目时，只需说：
- "帮我写一个PRD"
- "我需要为[产品想法]创建PRD"
- "开始PRD生成流程"

Skill会自动引导您完成整个思考流程，最终生成可直接使用的PRD文档。
