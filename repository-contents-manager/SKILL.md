# Repository Contents Manager - 仓库内容管理

## 技能概述

专门用于管理GitHub仓库文件结构和内容的Skill，帮助用户优化代码组织和文件管理。

## 主要功能

- 分析仓库文件结构
- 优化目录组织
- 管理大文件和二进制文件
- 代码质量检查
- 文件权限管理

## 使用场景

- 代码仓库重构
- 文件组织结构优化
- 大文件管理
- 代码质量监控
