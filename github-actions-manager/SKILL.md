# GitHub Actions Manager - 自动化工作流管理

## 技能概述

专门用于管理GitHub Actions工作流的Skill，帮助用户创建、配置和优化CI/CD流水线。

## 主要功能

- 创建工作流配置文件
- 配置触发条件和环境变量
- 管理secrets和权限
- 监控工作流执行状态
- 优化工作流性能

## 使用场景

- 自动化测试和部署
- 代码质量检查
- 容器构建和推送
- 多环境部署管理
