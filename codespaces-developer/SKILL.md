# Codespaces Developer - 云端开发环境管理

## 技能概述

专门用于管理GitHub Codespaces开发环境的Skill，帮助用户快速创建和配置云端开发环境。

## 主要功能

- 创建和配置Codespaces环境
- 管理开发容器配置
- 优化开发环境性能
- 管理扩展和工具链
- 监控资源使用情况

## 使用场景

- 快速搭建开发环境
- 团队开发环境标准化
- 多项目环境管理
- 开发工具链配置
