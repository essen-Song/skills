# Artifact Metadata Analyzer - 制品元数据管理

## 技能概述

专门用于分析和优化制品（Artifact）元数据的Skill，帮助用户管理构建产物和依赖关系。

## 主要功能

- 分析制品依赖关系
- 优化元数据存储结构
- 管理制品版本控制
- 监控制品生命周期
- 优化存储空间使用

## 使用场景

- 容器镜像管理
- 软件包版本控制
- 构建产物优化
- 依赖关系分析
