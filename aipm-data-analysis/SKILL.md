# AI产品数据分析 - 数据洞察助手

## 技能概述

帮助AI产品经理进行数据分析和洞察的工具，支持用户行为分析、产品指标监控和实验效果评估。

## 主要功能

- **用户行为分析**：分析用户使用模式和偏好
- **产品指标监控**：跟踪关键产品指标变化
- **A/B测试分析**：评估功能实验的效果
- **数据可视化**：创建直观的数据图表和报告
- **洞察发现**：从数据中发现产品改进机会

## 使用场景

- 产品功能效果评估
- 用户行为模式分析
- 产品迭代决策支持
- 数据驱动的产品优化
